# مشروع Andid HR Yemen الكامل

هذا المشروع هو موقع إلكتروني لنشر الوظائف والمناقصات في اليمن، يحتوي على واجهات متعددة مثل:  
- تسجيل الدخول  
- لوحة تحكم صاحب العمل  
- لوحة تحكم الإدارة  
- رفع واستعراض الوظائف والسير الذاتية

تم بناء المشروع باستخدام تقنيات React، Firebase، HTML، CSS، وجافاسكريبت.

---
